# Multi-Language Blog Example

This is a multi-language example of the `next-md-blog` library, demonstrating how to set up a blog with internationalization (i18n) support.

## Features

- **Multi-language support**: Blog posts organized by locale (en, fr)
- **Locale-based routing**: Routes like `/en/blog/welcome` and `/fr/blog/welcome`
- **Locale switcher**: Dropdown component to switch between languages while preserving the current route
- **Automatic static generation**: All blog posts are pre-rendered at build time
- **SEO optimized**: Includes metadata, structured data, and OG images
- **Tailwind CSS v4**: Uses OKLCH colors and `@tailwindcss/typography` plugin
- **Light & dark mode**: Theme toggle with system preference support
- **Type-safe**: Full TypeScript support
- **Beautiful UI**: shadcn/ui components with responsive design

## Project Structure

```
app/
  [locale]/              # Locale-based routing
    blog/
      [slug]/
        page.tsx         # Individual blog post page
        opengraph-image.tsx  # OG image generation
    blogs/
      page.tsx           # Blog listing page
    layout.tsx           # Locale layout wrapper
posts/
  en/                    # English blog posts
    welcome.md
  fr/                    # French blog posts
    welcome.md
```

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run the development server**:
   ```bash
   npm run dev
   ```

3. **Visit the blog**:
   - English: http://localhost:3000/en/blog/welcome
   - French: http://localhost:3000/fr/blog/welcome
   - Blog listing: http://localhost:3000/en/blogs or http://localhost:3000/fr/blogs

## How It Works

### Locale-based Routing

The app uses Next.js App Router with a `[locale]` dynamic segment to handle multiple languages:

- `/en/blog/[slug]` - English blog posts
- `/fr/blog/[slug]` - French blog posts
- `/en/blogs` - English blog listing
- `/fr/blogs` - French blog listing

### Content Organization

Blog posts are organized by locale in the `posts` directory:

```
posts/
  en/
    welcome.md
    another-post.md
  fr/
    welcome.md
    another-post.md
```

Each locale folder contains markdown files with frontmatter metadata.

### Creating New Posts

1. Create a markdown file in the appropriate locale folder:
   ```bash
   # English post
   posts/en/my-new-post.md
   
   # French post
   posts/fr/my-new-post.md
   ```

2. Add frontmatter:
   ```markdown
   ---
   title: "My New Post"
   date: "2024-01-15"
   description: "A description of my post"
   ---
   
   # My New Post
   
   Content goes here...
   ```

3. The post will automatically be available at:
   - `/en/blog/my-new-post`
   - `/fr/blog/my-new-post`

## Configuration

The blog is configured via `next-md-blog.config.ts`:

```typescript
export default {
  siteName: 'Multi-Language Blog',
  siteUrl: process.env.NEXT_PUBLIC_SITE_URL || 'https://example.com',
  defaultAuthor: 'Example Author',
  twitterHandle: '@example',
  defaultLang: 'en',
};
```

## Styling

This example uses **Tailwind CSS v4** with **OKLCH colors** and the **@tailwindcss/typography** plugin:

- **Color System**: OKLCH format for better color consistency (see `app/globals.css`)
- **Typography**: Automatic markdown styling with `prose` classes
- **Theme Support**: Light and dark modes with system preference
- **Locale Switcher**: Located in the header, allows switching between locales

### Customizing Colors

Update the color variables in `app/globals.css`:

```css
:root {
  --primary: hsl(142 76% 36%);  /* Your primary color */
  /* ... */
}

.dark {
  --primary: hsl(142 76% 36%);  /* Dark mode primary */
  /* ... */
}
```

## Adding More Locales

To add more locales:

1. Update the `locales` array in `app/[locale]/blog/[slug]/page.tsx`:
   ```typescript
   const locales = ['en', 'fr', 'es', 'de'];
   ```

2. Create a new folder in `posts/` for the locale:
   ```bash
   mkdir posts/es
   ```

3. Add blog posts in that locale folder.

## Generated by CLI

This example was created using the `next-md-blog-init` CLI with the following command:

```bash
node ../../dist/cli/cli.js \
  --non-interactive \
  --i18n-enabled \
  --locales="en,fr" \
  --content-dir="posts" \
  --blog-route="blog" \
  --blogs-route="blogs" \
  --example-post \
  --blog-pages \
  --og-image \
  --site-name="Multi-Language Blog" \
  --site-url="https://example.com" \
  --author="Example Author" \
  --twitter="@example"
```
